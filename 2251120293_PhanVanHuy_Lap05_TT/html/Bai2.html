1. Tương tác với HTML và CSS:
    • Thay đổi nội dung và cấu trúc HTML: JavaScript có thể thay đổi nội dung, thuộc tính và cấu trúc của các phần tử HTML.
    • Thay đổi kiểu dáng CSS: JavaScript có thể thay đổi các thuộc tính CSS để thay đổi giao diện của trang web.
  2. Xử lý sự kiện:
    • JavaScript có thể phản hồi các sự kiện như nhấp chuột, nhập liệu, di chuột, cuộn trang, và nhiều sự kiện khác.
  3. Xử lý dữ liệu:
    • Gửi và nhận dữ liệu:JavaScript có thể gửi và nhận dữ liệu từ máy chủ bằng AJAX hoặc Fetch API mà không cần tải lại trang.
    • Xử lý và thao tác với dữ liệu: JavaScript có thể xử lý dữ liệu từ các nguồn khác nhau, bao gồm cả JSON và XML.
  4. Tạo và kiểm soát hoạt ảnh
    • JavaScript có thể tạo ra các hoạt ảnh phức tạp và điều khiển các khung hình của chúng.
  5. Lưu trữ cục bộ:
    • Local Storage và Session Storage: JavaScript có thể lưu trữ dữ liệu trên trình duyệt của người dùng mà không cần sử dụng cookie.
  6. Xây dựng ứng dụng một trang (SPA):
    • Các framework và thư viện như React, Angular và Vue.js cho phép xây dựng các ứng dụng một trang hiện đại và tương tác.
  7. Xử lý đồng thời và công việc nền:
    • Web Workers: JavaScript có thể chạy các tác vụ trong nền mà không làm chậm trang web chính.
    • Promises và Async/Await: JavaScript có các công cụ mạnh mẽ để xử lý các tác vụ bất đồng bộ.
  8. Tương tác với API của trình duyệt:
    • JavaScript có thể sử dụng các API của trình duyệt như Geolocation (định vị địa lý), Web Storage, IndexedDB (cơ sở dữ liệu trong trình duyệt), và nhiều API khác.
  9. Tạo và điều khiển các yếu tố đa phương tiện:
    • JavaScript có thể điều khiển các yếu tố âm thanh và video, bao gồm việc phát, tạm dừng và thay đổi âm lượng.
